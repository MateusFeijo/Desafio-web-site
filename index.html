Surgimento da programação

(inicio)

A história da programação tem raízes na concepção dos primeiros dispositivos de cálculo mecanizado, mas a programação de computadores como a conhecemos hoje surgiu nos anos 1950.

As linguagens de programação são convenções e regras que permitem passar instruções para um computador através de palavras e símbolos.

A primeira linguagem de programação foi criada por Ada Lovelace em 1843. Ela usou papel para escrever o primeiro algoritmo para um computador, pois não havia computadores na época. Desde então, muitas linguagens de programação foram desenvolvidas e usadas para diferentes propósitos.

As linguagens de programação são linguagens usadas para a comunicação com o computador. Estas linguagens são constituídas de comandos, que quando utilizados corretamente, executam uma ação. A programação nos computadores não tem uma data correta de início. Tudo começou na década de 30, com os primeiros computadores elétricos. Em 1948, Konrad Zuse publicou sua criação, a linguagem de programação Plankalkül. Na época, ainda não tinha muita utilidade, então foi esquecida. Antes da programação passar para o computador, eram usados cartões de papelão que eram perfurados, criando códigos.

Foi na década de 50 que as primeiras linguagens modernas surgiram. FORTRAN (1955), LISP e COBOL. Tambem apareceu na mesma epoca a ALGOL 60.

Como funciona

(inicio)

A programação é um processo que envolve a criação de programas por meio de uma linguagem de programação, que é composta por um conjunto de regras, símbolos, palavras-chave e números. O programador, ou desenvolvedor, é quem escreve os códigos em uma linguagem de programação, que são transformados em um software.

A programação é baseada em quatro pilares do pensamento computacional: Decomposição, reconhecimento de padrões, abstração e algoritimo.

A linguagem de programação é uma forma de comunicação entre o programador e o computador, que permite criar ordens para que a máquina as execute. Os códigos-fonte criados pelos programadores são uma sequência de comandos que indicam ao computador o que fazer, como mostrar informações na tela, criar aplicativos, sites e programas de software.

Não existe uma graduação específica para se tornar um programador, mas algumas graduações podem ajudar na trajetória profissional, como Engenharia da Computação, Ciência da Computação, Análise e Desenvolvimento de Sistemas, entre outras.

O que são linguagens de programação

(inicio)

Linguagens de programação são sistemas de comunicação que permitem a interação entre humanos e máquinas, por meio de um conjunto de regras, símbolos, palavras-chave e comandos. Elas são usadas para criar programas, aplicativos, sites, e até mesmo TVs.

Existem diferentes tipos de linguagens de programação, como as de baixo nível, que são mais orientadas para a máquina, e as de alto nível, que são mais fáceis de usar. Uma das linguagens de programação mais utilizadas é o Java, que é usado para criar aplicativos, servidores web, ferramentas de software, entre outros

A linguagem de programação é um método padronizado, formado por um conjunto de regras sintáticas e semânticas, de implementação de um código fonte - que pode ser compilado e transformado em um programa de computador, ou usado como script interpretado - que informará instruções de processamento ao computador.

Lista de programas mais utilizados
(inicio)
  1. JavaScript está em primeiro lugar na lista do Stack Overflow há seis edições! Essa linguagem de programação é utilizada de maneira ampla para aplicações web e tem ganhado espaço tanto no desktop, como no mobile. É bastante utilizado para criar interatividade. Mesmo sendo uma linguagem um pouco mais antiga em comparação com a maioria, ainda é bastante requisitada por sua simplicidade.
  2. SQL é o padrão para a manipulação de bancos de dados através de SGBDs (sistema de gerenciamento). Com essa linguagem, é possível realizar diversas tarefas como criar arquivos no banco de dados, consultar informações, inserir e alterar registros, gerenciar acesso de usuários, entre outros fatores.
  3. Java também está subindo nos índices de popularidade de linguagem. Ela é utilizada para a criação de aplicativos Android, o sistema operacional mais presente nos aparelhos móveis. Além disso, entre seus pontos positivos estão a escalabilidade, sua adaptação com o crescimento de utilizações do software e a retrocompatibilidade. Uma versão antiga do código continua sendo reconhecida por versões mais recentes.
